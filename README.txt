Pour exécuter la première version du jeu, il faut exécuter Menu.py.

Il faut ensuite sélectionner son niveau en cliquant sur les boutons à l'aide de la souris.
Pour déplacer les molécules, il faut les sélectionner et les translater à l'aide de la souris.
ATTENTION : Pour le moment, les déplacement sont libres (pas encore de solution pour controler mouvements).

Attention également à bien sélectionner une seule molécule à la fois. Ls hitbox sont des gros rectangles, il est possible qu'une hitbox soit dans une autre hitbox.
Par exemple, pour le level easy : le rouge est dans le violet, donc il faut d'abord déplacer le violet, puis le rouge.

Le level se termine lorsque la molécule rouge atteint la sortie du plateau. Lorsqu'un niveau se termine, le prochain commence.
Si le dernier niveau est terminé, le programme s'arrête.

Pour le moment, beaucoup de bugs sont encore présent, et je n'ai aucune solution pour y remédier.
